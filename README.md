**DevOps Notes**

Making a new service:
- use one of the template repos provided in our org
- replace all of the necessary information:
  - package.json:
    - name
    - version
    - description
  - chart/Chart.yaml:
    - name
    - version
    - appVersion
- create a .env file for dev env variables
